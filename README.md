# EcoBricks Initiative — 5‑Page Website

## Quick Start (GitHub Pages)
1. Create a **public** GitHub repo named `ecobricks-site` (or any name).
2. Upload all files in this folder to the repo root.
3. Go to **Settings → Pages** → Source: **Deploy from a branch**, Branch: **main**, Folder: **/**.
4. Your site will appear at `https://USERNAME.github.io/REPO` within a couple minutes.

## Pages
- `index.html` — Home / value & CTA
- `what-are-ecobricks.html` — Definition & benefits
- `how-to.html` — Step‑by‑step guide
- `community.html` — Project types, timeline, impact
- `contact.html` — Privacy‑friendly form
